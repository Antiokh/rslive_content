Руководство для владельцев жилья

С помощью портала еУправа арендодатель может зарегистрировать иностранца онлайн, не посещая полицию. Ниже описаны шаги для регистрации и необходимые данные.

## Доступ к eUprava
  * Скачайте приложение ConsentID.
  * Посетите почтовое отделение для привязки личных документов и получения QR-кода.
  * Считайте QR-код через приложение для активации.

## Регистрация иностранца
  - Перейдите на сайт [еУправа](https://euprava.gov.rs/usluge/4494) и войдите через приложение <autott>adaptation:consentid|ConsentID</autott>.
  - Нажмите "Покрени услугу"
  - Подтвердите доступ и выберите опцию «Покрени услугу».
  - Введите данные иностранца, проверьте их и подайте заявку.
  - «[Белый картон](/arrival/beli-karton/)» отправится арендодателю почти мгновенно, а гостю может прийти позже.

:::note
**Примечание:** Лучше использовать e-mail на Gmail, так как документы не отправляются на mail.ru и yandex.ru.
:::
