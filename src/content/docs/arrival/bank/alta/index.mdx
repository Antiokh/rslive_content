[Alta Banka](https://altabanka.rs) — сербский банк, активно работающий с иностранцами и предпринимателями. Банк вырос из обменной сети и сейчас занимает сильные позиции в эквайринге и карточном бизнесе. Имеет широкую сеть банкоматов через партнёрство с MoneyGet.

В головном офисе в Белграде есть сотрудники, говорящие по-русски.

-----

## Шаги для открытия счета

  * **Документы (физлица)**:
    * Загранпаспорт.
    * Российский паспорт.
    * <autott>arrival:beli-karton|Белый картон</autott>.
    * Основание пребывания (ВНЖ, договор аренды, контракт, бизнес).
    * Подтверждение источника средств (выписка / трудовой договор).

  * **Документы (предприниматели)**:
    * Паспорт.
    * Выписка из АПР.
    * Решение налоговой.
    * Анкета банка.

⚠ Без подтверждения связи с Сербией могут открыть только RSD-счёт. Валютные счета требуют экономического обоснования.

Официальный список документов:
[необходимая документация](https://altabanka.rs/neophodna-dokumentacija/)

## Типы счетов

  * **Счет с основными услугами**
  * **Мультивалютный счет**
  * Онлайн-банкинг: [Alta Online](https://altabanka.24x7.rs/altaonline/Identity/Login)
  * Мобильное приложение: [мобильный банкинг](https://altabanka.rs/stanovnistvo/digitalno-usluge/mobilno/)

## Карты

  * Если открыт только RSD-счёт → выдают **DinaCard**
  * При наличии валютного счёта → **Visa Debit**
  * Поддержка 3D-Secure
  * Снятие без комиссии в банкоматах Alta Bank и MoneyGet
  * Через банкомат выдаётся только RSD (даже с валютного счёта)
  * Валюту можно снять в кассе отделения

## Тарифы (ориентир)

  * Физлица: ~280 RSD / месяц
  * Юрлица: ~500 RSD / месяц (при движении средств)

Актуальные тарифы:
[официальный прейскурант](https://altabanka.rs/tarife/)

## IPS / Prenesi

Банк подключён к системе мгновенных платежей NBS (IPS).

  * Переводы по номеру телефона (Prenesi)
  * Мгновенные RSD-переводы 24/7
  * Работает через мобильное приложение

-----

## Гарантия вкладов

Вклады физических лиц застрахованы в соответствии с законодательством Сербии.

  * Гарантия до **50 000 EUR** на одного вкладчика
  * Распространяется на депозиты в RSD и валюте
