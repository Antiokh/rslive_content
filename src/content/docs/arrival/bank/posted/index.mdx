Государственный банк, похожий по роли на Сбербанк или ПочтаБанк в России.

## Особенности

- вся документация на сербском  
- банковское резидентство не появляется автоматически после открытия счёта  
- иностранцам обычно выпускают только валютную карту  
- онлайн-переводы между своими счетами могут быть недоступны  
- много отделений по Белграду  
- по вопросам бизнес-счёта лучше обращаться в отделение, где счёт открывали  
- мобильное приложение ограничено по возможностям  
- SMS об операциях платные: 1 динар за SMS  

## Счёт физлица

Можно открыть нерезидентский валютный счёт с картой Visa.

[Online Banking](https://onlinebanking.posted.co.rs/webapp/Identity/Login)

### Документы для открытия

- заграничный паспорт  
- внутренний паспорт РФ  
- [белый картон](/arrival/beli-karton/) — даже если уже есть [РВП/ВНЖ](/arrival/boravak/)  
- [договор аренды](/move/housing/)  
- карточка РВП/ВНЖ, если есть  

### Сроки открытия

Около месяца. Обычно нужно прийти несколько раз.  
До выдачи пластиковой карты выдают временную карточку с данными счёта.

## Счёт юрлица

### Особенности

- изменения по счёту — в том офисе, где открывали  
- онлайн-банкинг без смарт-карты  
- много отделений → меньше очереди  

<Aside type="note" title="Вход в онлайн-банкинг юрлиц">
  [Онлайн-банкинг юрлица Поштанска Штедионица](https://webebank.ebb-bg.com/sbpo_mbank/auth/login)
</Aside>

### Открытие

#### Документы

- все документы как для личного счёта  
- выписка из АПР  
- решение от налоговой  

#### Процедура

- заполнение анкеты  
- подача заявления  
- ожидание месяц и более
