# Открытие общества с ограниченной ответственностью (D.O.O.)

Общество с ограниченной ответственностью (*Društvo sa ograničenom odgovornošću*, D.O.O.) — одна из основных форм ведения бизнеса в Сербии. Аналог российского ООО или европейского LLC.

D.O.O. подходит для:

- IT-компаний;
- агентств;
- e-commerce;
- консалтинга;
- международного бизнеса;
- деятельности с несколькими учредителями;
- случаев, когда требуется разделение личных и корпоративных финансовых рисков.

Регистрация осуществляется через [[/gov/apr/|Агентство по регистрации хозяйственных субъектов (АПР)]]. Основной закон — [[https://www.paragraf.rs/propisi/zakon_o_privrednim_drustvima.html|Закон о хозяйственных обществах]].

## Основные особенности D.O.O.

| Параметр | Значение |
|---|---|
| Минимальный уставный капитал | 100 RSD |
| Ответственность участников | В пределах вклада |
| Учредители | Физические и юридические лица |
| Директор | Может быть иностранцем |
| Бухгалтерия | Обязательна |
| НДС (PDV) | Обязателен после 8 млн RSD оборота за 12 месяцев |
| Налог на прибыль | 15% |
| Налог на дивиденды | 15% |

:::tip
Минимальный капитал в 100 динаров юридически допустим, однако банки, контрагенты и платёжные системы часто воспринимают такой капитал как формальность. На практике многие компании указывают более реалистичный капитал.
:::

## Отличие D.O.O. от предпринимателя (PR)

| D.O.O. | [[/arrival/business/prvsdoo/|Предприниматель (PR)]] |
|---|---|
| Отдельное юридическое лицо | Физлицо ведёт бизнес лично |
| Ограниченная ответственность | Возможна личная ответственность |
| Обязательная бухгалтерия | У паушальцев может быть упрощённый режим |
| Можно распределять дивиденды | Доход считается доходом физлица |
| Более сложное администрирование | Проще и дешевле |
| Подходит для инвесторов и партнёров | Ограничено масштабирование |

## Что потребуется для регистрации

Перед подачей заявления обычно подготавливают:

- название компании;
- юридический адрес в Сербии;
- данные учредителей;
- данные директора;
- виды деятельности (шифра деятельности / šifra delatnosti);
- учредительный акт;
- решение о назначении директора;
- паспорт учредителя и директора;
- адрес для получения почты;
- электронную подпись (*квалификovani elektronski sertifikat*) — если регистрация подаётся онлайн.

## Выбор названия компании

Название проверяется через реестр [[/gov/apr/|АПР]].

Требования:

- название должно отличаться от существующих;
- нельзя использовать вводящие в заблуждение названия;
- некоторые слова требуют дополнительных согласований;
- организационная форма обычно добавляется в конце: `D.O.O.`.

Проверить доступность можно через официальный поиск:

- [[https://pretraga2.apr.gov.rs/EnterpriseSearch/Search/Search|APR Pretraga]]

:::warning
Даже небольшое визуальное сходство с существующей компанией может привести к отказу в регистрации.
:::

## Юридический адрес

Для регистрации необходим адрес в Сербии.

Обычно используют:

- собственное помещение;
- аренду;
- virtual office;
- coworking с юридическим адресом.

На адрес будет приходить официальная корреспонденция от:

- [[/gov/apr/|АПР]];
- [[/gov/purs/|Налоговой службы]];
- банков;
- инспекций.

## Учредительные документы

Если учредитель один — оформляется:

- *Odluka o osnivanju* (решение о создании).

Если учредителей несколько:

- *Ugovor o osnivanju* (договор об учреждении).

Обычно документ содержит:

- название компании;
- адрес;
- виды деятельности;
- размер капитала;
- доли участников;
- директора;
- порядок принятия решений.

## Виды деятельности (шифра деятельности)

Основной вид деятельности выбирается по классификатору:

- [[https://www.stat.gov.rs/sr-latn/oblasti/klasifikacije/nkd10/|Klasifikacija delatnosti]]

Для IT-компаний часто используют:

| Код | Деятельность |
|---|---|
| 6201 | Разработка ПО |
| 6202 | IT-консалтинг |
| 6311 | Обработка данных |
| 7022 | Бизнес-консалтинг |

:::tip
Дополнительные виды деятельности можно добавить позже через изменение данных в АПР.
:::

## Подача заявления в АПР

Регистрация возможна:

- лично;
- через представителя;
- онлайн с электронной подписью.

Используется:

- единая регистрационная форма;
- учредительные документы;
- подтверждение оплаты пошлины.

Официальная инструкция АПР:

- [[https://www.apr.gov.rs/registri/privredna-drustva/uputstva.2035.html|Uputstva APR]]

## Госпошлины

Размеры пошлин периодически меняются. Актуальные тарифы публикуются АПР:

- [[https://www.apr.gov.rs/registri/privredna-drustva/naknade.2029.html|Тарифы АПР]]

Обычно оплачиваются:

- регистрация D.O.O.;
- публикация;
- дополнительные изменения при необходимости.

## Срок регистрации

Обычно регистрация занимает:

- 1–5 рабочих дней после подачи корректного пакета документов.

Если АПР обнаружит ошибки, выносится:

- *rešenje o odbačaju* — отказ с возможностью исправления.

## Получение ПИБ и регистрации

После регистрации компания получает:

- матичный номер (*matični broj*);
- ПИБ (*PIB*);
- регистрацию в Центральном реестре;
- данные для налогового учёта.

Проверка компании:

- [[https://pretraga2.apr.gov.rs/EnterpriseSearch/Search/Search|APR Pretraga]]

## Открытие банковского счёта

После регистрации открывается корпоративный счёт.

Часто используются:

- [Raiffeisen Bank](/arrival/bank/raiffeisen/);
- [[/arrival/bank/aik/|AIK Banka]];
- [Poštanska štedionica](/arrival/bank/posted/);
- [[/arrival/bank/intesa/|Banca Intesa]].

Банк обычно запрашивает:

- решение АПР;
- ПИБ;
- паспорт директора;
- подтверждение адреса;
- образцы подписей;
- данные о бенефициарах.

:::warning
Для иностранных директоров compliance-проверка может занимать от нескольких дней до нескольких недель.
:::

## Регистрация бенефициаров (UBO)

После регистрации необходимо внести данные о конечных бенефициарных владельцах:

- [[https://www.apr.gov.rs/registri/centralna-evidencija-stvarnih-vlasnika/o-registru.2140.html|Центральный реестр бенефициаров]]

Срок подачи — обычно 15 дней с момента регистрации изменений.

За нарушение предусмотрены штрафы.

## Бухгалтерия

D.O.O. обязано вести полноценный бухгалтерский учёт.

Обычно бухгалтер отвечает за:

- зарплаты;
- налоги;
- отчётность;
- PDV;
- годовой баланс;
- взаимодействие с налоговой.

См. также:

- [[/arrival/business/accountant/|Бухгалтер в Сербии]]
- [[/arrival/business/porezi/|Налоги бизнеса]]

## Налогообложение D.O.O.

### Налог на прибыль

Ставка:

- 15% от прибыли.

Закон:

- [[https://www.paragraf.rs/propisi/zakon_o_porezu_na_dobit_pravnih_lica.html|Закон о налоге на прибыль юридических лиц]]

### НДС (PDV)

Стандартная ставка:

- 20%.

Льготная ставка:

- 10%.

Обязательная регистрация возникает при обороте более:

- 8 000 000 RSD за последние 12 месяцев.

Можно зарегистрироваться добровольно.

Подробнее:

- [[/arrival/business/pdv/|PDV (НДС) в Сербии]]

### Налог на дивиденды

При выплате дивидендов физлицам удерживается:

- 15%.

Для международных структур возможно применение соглашений об избежании двойного налогообложения.

## Зарплата директора

Директор D.O.O.:

- может быть трудоустроен;
- может работать без трудового договора в некоторых случаях;
- может быть иностранцем;
- часто используется как основание для [[/arrival/boravak/work-permit/|ВНЖ по работе]].

См. также:

- [[/arrival/radna-dozvola/|Разрешение на работу]]
- [Временное пребывание](/arrival/boravak/)

## Электронные сервисы

После открытия компании обычно подключают:

- [[/integration/euprava/|eUprava]];
- [[/integration/edostava/|eDostava]];
- [[/arrival/business/eporezi/|ePorezi]];
- электронный сертификат;
- электронную подпись.

## Типичные расходы после открытия

| Расход | Примерный диапазон |
|---|---|
| Бухгалтер | от 50–300+ EUR/мес |
| Банк | от 0–50 EUR/мес |
| Зарплатные налоги | зависят от схемы |
| Электронная подпись | периодически продлевается |
| Юрадрес | зависит от адреса |

## Частые ошибки

- регистрация без понимания налоговой нагрузки;
- выбор неподходящего вида деятельности;
- отсутствие бухгалтера;
- использование фиктивного адреса;
- пропуск подачи отчётности;
- отсутствие регистрации бенефициара;
- открытие D.O.O. вместо более простого [[/arrival/business/prvsdoo/|PR]].

## Когда D.O.O. обычно лучше PR

D.O.O. часто выбирают если:

- есть несколько партнёров;
- нужны инвестиции;
- требуется ограничение ответственности;
- работа идёт с иностранными контрагентами;
- необходима более «корпоративная» структура;
- планируется продажа долей;
- обороты существенно выше паушального лимита.
