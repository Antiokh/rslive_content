## Когда требуется

- подача документов на ВНЖ и РВП  
- оформление брака  
- нострификация диплома  
- взаимодействие с госорганами  

## Важно

<Aside type="caution" title="Юридический статус перевода">
  Только судебный переводчик (sudski tumač) имеет право заверять перевод документов для официального использования в Сербии.
</Aside>
