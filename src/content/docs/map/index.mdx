## PlanPlus

В Сербии есть собственный картографический сервис [PlanPlus](https://www.planplus.rs/en/). В этой энциклопедии он используется как основной локальный источник картографической информации.

### Недостатки

- мало отзывов у объектов  
- мало фотографий  
- нет микронавигации внутри сложных объектов  

### Преимущества

- подробные карты по Сербии  
- качественные маршруты, включая общественный транспорт  
- тематические подборки объектов

## Google Карты

Google Карты подходят для повседневного использования: поиск мест, отзывы, маршруты.  
См. также: [общественный транспорт](/arrival/prevoz/)

### Преимущества

- хорошие автомобильные маршруты  
- отображение пробок  
- большая база заведений и отзывов  
- интеграция с транспортом  

### Недостатки

- неточная картография домов  
- устаревшие данные по учреждениям  
- хуже работает с кириллицей  

---

## Яндекс Карты

Подходят для русскоязычных пользователей, но имеют ограничения в Сербии.

### Преимущества

- удобный общественный транспорт  
- русскоязычный интерфейс  
- интеграция с Яндекс Go  

### Недостатки

- слабая актуальность вне крупных городов  
- проблемы с латиницей  
- устаревшие данные  
- неточности в адресах
