# ЭЦП, картицы, читач, челик


## Чтение данных с личных карт

Водительские права (*vozačka dozvola*), карточка ОМС (*zdravstvena kartica*) и ID (*lična karta*) имеют чип и являются смарт-картами. Их можно прочитать через специальный считыватель.

С февраля 2024 года при подаче некоторых заявлений требуется прикладывать распечатку данных личной карты — *očitana lična karta*. Поскольку данные обычно не меняются, их можно считать один раз и сделать нужное количество копий.

Считыватель смарт-карт также может использоваться:

- для ЭЦП в онлайн-банкинге юрлица в [Alta Banka](/arrival/bank/alta/);
- для входа и подписания на некоторых государственных сайтах;
- для работы с квалифицированным электронным сертификатом на личной карте.

## Как считать данные

### Вариант 1: в копирнице

Многие фотокопирницы имеют считыватель смарт-карт и могут выдать распечатку данных — *očitana kartica*.

### Вариант 2: самостоятельно

<Steps>
1. Купите считыватель смарт-карт:
   - [компактный считыватель Samtec SMT-610](https://gigatron.rs/proizvod/samtec-smart-card-reader-smt-610-8606108800416);
   - [другие считыватели карт](https://gigatron.rs/oprema-za-racunare/smestanje-podataka/citaci-kartica).

2. Установите программу для чтения карты:
   - [официальная страница МВД](http://mup.gov.rs/wps/portal/sr/gradjani/dokumenta/licna%20karta/licna%20karta%20aplikacija/!ut/p/z0/04_Sj9CPykssy0xPLMnMz0vMAfIjo8zi_S19zQzdDYy8LUwMDA0czRzdLQxMLYwMgg31g4uL9L30o_ArAppiVOTr7JuuH1WQWJKhm5mXlq8fkZOZnJeokJ1YVJKIwlFILMjJzE5MzsxK1C_Idg93dFRUBABugbrQ/);
   - [Челик](http://ca.mup.gov.rs/ca/ca_cyr/start/kes/#collapseOne5)[^1] — официальная программа, читает личную карту;
   - [TrustEdge](http://ca.mup.gov.rs/ca/ca_cyr/start/kes/#collapseOne1) — для ЭЦП, требуется заявление в полицию;
   - [баш-челик](https://github.com/ubavic/bas-celik/releases)[^2] — неофициальная программа, читает несколько видов карт, поддерживает английский язык, тёмную тему и Linux.
</Steps>

<Aside type="caution" title="Официальность ПО">
  Для юридически значимых действий используйте официальные инструменты и инструкции МВД. Неофициальные программы удобны для чтения данных, но ответственность за их использование остаётся на пользователе.
</Aside>

## Получение и использование ЭЦП на личной карте

Квалифицированный электронный сертификат можно записать на личную карту с чипом.

[Информация на официальном сайте МВД](http://ca.mup.gov.rs/ca/ca_cyr/start/kes/)

<Aside type="tip" title="Запись на приём">
  Рекомендуется записаться на приём для получения квалифицированного электронного сертификата через электронный сервис портала eUprava.
</Aside>

## Предварительные условия

Для получения квалифицированного электронного сертификата нужно иметь действительное удостоверение личности с чипом.

Если пользователь забыл ПИН-код, при подаче заявления на получение сертификата нужно также подать запрос на разблокировку удостоверения личности с изменением ПИН-кода.

### Термины

- **Пользователь** — физическое лицо, которому предоставляется услуга выдачи квалифицированного электронного сертификата.
- **Регистрационное подразделение** — подразделение МВД, уполномоченное на регистрацию пользователей и выдачу удостоверений личности.
- **Активационные данные** — пользовательский ПИН-код, который выдаётся в защищённом конверте при получении удостоверения личности с чипом.

## Выдача квалифицированного электронного сертификата

<Steps>
1. Подготовьте заявление на выдачу или обновление квалифицированного электронного сертификата — форма 1. Для выдачи сертификата выберите пункт **«Выдача сертификата»**.

2. Подготовьте два экземпляра подписанных общих условий предоставления услуги выдачи квалифицированных электронных сертификатов.

3. Передайте заявление и документы сотруднику регистрационного подразделения.

4. При необходимости подпишите договор о предоставлении услуги выдачи квалифицированного электронного сертификата.

5. Если нужно включить e-mail в сертификат, сотрудник введёт его в предусмотренное поле.

6. Введите ПИН-код самостоятельно. ПИН известен только пользователю.

7. После записи сертификата на чип сотрудник возвращает удостоверение личности и один экземпляр документов.
</Steps>

## Обновление сертификата

Обновление возможно, если:

- срок действия сертификата истёк;
- до окончания срока действия осталось шесть месяцев или меньше.

Для обновления нужен действительный документ с чипом и действующий договор.

<Steps>
1. Заполните заявление на обновление сертификата — форма 1. Выберите пункт **«Обновление сертификата»**.

2. Передайте заявление сотруднику регистрационного подразделения.

3. Если нужно включить e-mail в сертификат, сотрудник введёт его в предусмотренное поле.

4. Введите ПИН-код самостоятельно.

5. После записи обновлённого сертификата сотрудник возвращает удостоверение личности.
</Steps>

## Разблокировка ПИН-кода

ПИН — секретный активационный код, который пользователь получает в защищённом конверте при выдаче удостоверения личности с чипом.

ПИН используется:

- при выдаче квалифицированного электронного сертификата;
- при каждом применении сертификата.

Если трижды ввести неверный ПИН-код, ПИН блокируется.

### Условия разблокировки

Для разблокировки нужно:

- действующее удостоверение личности;
- личное обращение;
- заявление о разблокировке ПИН-кода.

<Steps>
1. Заполните заявление на разблокировку ПИН-кода удостоверения личности — образец 2.

2. Укажите, нужно сохранить существующий ПИН или изменить его.

3. Передайте заявление сотруднику регистрационного подразделения.

4. Если выбрано изменение ПИН-кода, задайте новый ПИН самостоятельно.

5. После успешной разблокировки сотрудник возвращает удостоверение личности.
</Steps>

## Изменение статуса сертификата

Пользователь может запросить изменение статуса сертификата на основании действующего договора и закона об электронном документе, электронной идентификации и доверительных услугах.

Статус сертификата может быть:

- **активный**;
- **отозванный**;
- **приостановленный**.

Сертификат со статусом **«приостановленный»** может быть реактивирован и возвращён в статус **«активный»**.

## Отзыв сертификата

<Steps>
1. Заполните заявление на изменение статуса сертификата.

2. Выберите пункт **«Отзыв сертификата»**.

3. Передайте заявление сотруднику регистрационного подразделения.

4. Сотрудник запускает процедуру отзыва.

5. После отзыва сертификат аннулируется постоянно и добавляется в список отозванных сертификатов.
</Steps>

## Приостановка сертификата

<Steps>
1. Заполните заявление на изменение статуса сертификата.

2. Выберите пункт **«Приостановка сертификата»**.

3. Передайте заявление сотруднику регистрационного подразделения.

4. Сотрудник запускает процедуру приостановки.

5. Сертификат временно прекращает действие и добавляется в список отозванных сертификатов.
</Steps>

## Реактивация сертификата

<Steps>
1. Заполните заявление на изменение статуса сертификата.

2. Выберите пункт **«Реактивация сертификата»**.

3. Передайте заявление сотруднику регистрационного подразделения.

4. Сотрудник запускает процедуру реактивации.

5. Сертификат удаляется из списка отозванных сертификатов и возвращается в статус **«активный»**.
</Steps>
