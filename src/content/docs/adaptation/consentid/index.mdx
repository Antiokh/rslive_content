**ConsentID** — официальное мобильное приложение для подтверждения личности при входе в государственные онлайн-сервисы Сербии через [eID](/adaptation/eid/).

Приложение используется для доступа к [eUprava](/integration/euprava/), eHealth, eTaxes и другим государственным сервисам.

## Основные функции

- **Безопасная аутентификация** — подтверждение личности при входе в государственные сервисы.
- **Электронная подпись** — использование квалифицированной электронной подписи в облаке.
- **Двухфакторная защита** — вход подтверждается через приложение и уникальные коды.

## Установка и активация

<Steps>
1. Зарегистрируйтесь на [портале eID](/adaptation/eid/).

2. Скачайте приложение:
   - [Google Play](https://play.google.com/store/apps/details?id=nl.aeteurope.mpki.gui)
   - [App Store](https://apps.apple.com/us/app/consentid/id883224643)

3. Посетите [аккредитованное почтовое отделение](/media/adaptation/lokacije_eid.pdf) для активации.

4. Получите QR-код или код активации.

5. Отсканируйте QR-код в приложении ConsentID или введите код вручную.
</Steps>

<Aside type="caution" title="QR-код одноразовый">
  После сканирования QR-код перестаёт быть действительным.
</Aside>

## После активации

После активации ConsentID можно использовать для входа в аккаунт [eID](/adaptation/eid/) и связанные государственные сервисы:

- [eUprava](/integration/euprava/)
- eHealth
- eTaxes
- другие сервисы, подключённые к eID

## Преимущества

- вход без смарт-карты и кардридера;
- доступ с телефона;
- подтверждение личности онлайн;
- более безопасный вход по сравнению с обычным паролем.
