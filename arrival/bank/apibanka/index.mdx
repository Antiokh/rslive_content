[API Banka a.d. Beograd](https://www.apibank.rs) — сербский коммерческий банк, работающий с иностранцами, включая граждан РФ.

Банк участвует в системе страхования вкладов Республики Сербия: гарантия составляет до 50 000 EUR на одного вкладчика.[^1]

Известен более лояльным отношением к нерезидентам, но имеет повышенные комиссии по сравнению с большинством банков Сербии.

## Открытие счёта для физических лиц

### Документы

- Загранпаспорт
- [Белый картон](/arrival/beli-karton/)
- Подтверждение источника средств, если банк запросит
- Анкета банка

Требуется личная идентификация в отделении.[^2]

Минимальный первоначальный взнос отсутствует.[^3]

## Стоимость для физических лиц

| Услуга | Стоимость |
| --- | --- |
| Открытие счёта для нерезидента | ~6000 RSD единовременно[^12] |
| Месячное обслуживание для нерезидента | ~1200 RSD / мес[^13] |
| Месячное обслуживание для резидента | ~150 RSD / мес[^14] |

## Открытие счёта для предпринимателей и компаний

### Основные документы

- Выписка из АПР, не старше 3 месяцев
- Учредительные документы
- Карточка депонированных подписей
- Паспорта директора и бенефициаров с долей 25%+
- Заявление банка

Требования опубликованы на официальной странице корпоративных счетов.[^4]

### Стоимость для бизнеса

| Услуга | Стоимость |
| --- | --- |
| Открытие счёта для компании из ЕС | ~1000 EUR[^15] |
| Открытие счёта для компании вне ЕС | ~1500 EUR[^15] |
| Месячное обслуживание | 50–60 EUR[^16] |

> Это одни из самых высоких тарифов среди сербских банков.

## Счета

- RSD-счёт
- Мультивалютный счёт: EUR, USD и другие валюты
- Сберегательные счета
- Срочные депозиты

Описание мультивалютных счетов опубликовано на сайте банка.[^5]

## Банковские карты

API Banka выпускает карты системы **Visa** и национальной системы **DinaCard**.[^6]

### Дебетовые карты

- Visa Classic Debit
- Visa Infinite Debit
- DinaCard

DinaCard:

- Бесплатный выпуск
- Бесплатное годовое обслуживание
- Использование только в Сербии[^7]

Visa:

- Использование за границей
- Онлайн-платежи
- Конвертация валюты около 3%[^8]

## Онлайн-банкинг

API Banka предоставляет:

- Web-банкинг
- Мобильное приложение API mBank

Описание сервиса опубликовано на сайте банка.[^9]

Функции:

- Переводы внутри Сербии
- Международные переводы
- Обмен валют
- Депозиты
- Просмотр выписок

## SWIFT и международные переводы

SWIFT: **APIBRSBG**

### Исходящие переводы

- 0,8% от суммы
- Минимум около 1500–3000 RSD
- Дополнительно оплачиваются расходы банков-корреспондентов

Подтверждение: тарифы для нерезидентов.[^10]

### Входящие переводы

- 0,2–0,3% от суммы
- RUB-поступления могут облагаться комиссией до 3%

Подтверждение: тарифы для нерезидентов.[^11]

## Плюсы и минусы

| Плюсы | Минусы |
| --- | --- |
| Открывают счета нерезидентам | Высокие комиссии |
| Русскоязычные сотрудники | Дорогие SWIFT-переводы |
| Мультивалютные счета | Комиссия за конвертацию |
| Visa-карты | Платное обслуживание |
| Страхование вкладов | Дорогие корпоративные счета |

## Когда выбирать API Banka

- Если нужен счёт без ВНЖ
- Если другие банки отказали
- Если операции по карте и SWIFT редкие

## Когда лучше выбрать другой банк

- При частых международных переводах
- При активном использовании карты за границей
- Если важны низкие комиссии
